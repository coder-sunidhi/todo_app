# Todo List App

A simple console-based To-Do List application built in Java. This application allows users to add, view, and remove tasks, with persistent storage using a text file.

## Features

- **View Tasks**: Display all current tasks with numbering.
- **Add Task**: Add new tasks to the list.
- **Remove Task**: Delete a specific task by its number.
- **Persistent Storage**: Tasks are automatically saved to and loaded from `tasks.txt` file.
- **User-friendly Console Interface**: Simple menu-driven interaction.

## Technologies Used

- Java (JDK)
- Standard Java I/O (File handling with BufferedReader/Writer)
- ArrayList for in-memory task storage

## Prerequisites

- Java Development Kit (JDK) installed on your system (Java 8 or higher recommended).
- A terminal or command prompt to run the application.
